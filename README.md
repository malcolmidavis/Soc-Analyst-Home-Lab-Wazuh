# SOC Analyst Home Lab - Wazuh

## Overview

This project documents the deployment of a personal Security Operations Center (SOC) home lab using Wazuh on Ubuntu Server and a Windows 11 ARM endpoint running inside UTM on Apple Silicon.

The purpose of this lab is to gain practical experience deploying a SIEM, managing Linux servers, onboarding Windows endpoints, collecting security events, and building real-world SOC analyst skills.
